# İnternet Cafe Yönetim Sistemi

ASP.NET Core MVC ile geliştirilmiş, 4 katmanlı mimari yapısında internet cafe yönetim uygulaması.

## Özellikler
- Kullanıcı yönetimi (CRUD)
- Bilgisayar yönetimi (Durum takibi)
- Oturum başlatma/bitirme
- Otomatik ücret hesaplama
- Müsait bilgisayar filtreleme

## Teknolojiler
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Bootstrap

## Katmanlar
- **Varliklar** - Entity sınıfları
- **VeriKatmani** - DbContext, veritabanı işlemleri
- **IsKatmani** - Business logic, servisler
- **MVC** - Controller ve View'lar

## Not
Bu proje katmanlı mimari öğrenme sürecinde geliştirilmiştir.
